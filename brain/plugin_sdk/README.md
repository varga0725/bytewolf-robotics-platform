# Plugin SDK

ByteWolf Plugin SDK és capability registry. A workstream charter-e, contract-terve,
Definition of Done-ja és elfogadási kritériumai: [`docs/workstreams/plugin-sdk.md`](../../docs/workstreams/plugin-sdk.md).

Safety-határ: ez a réteg nem érheti el a MAVSDK/PX4 végrehajtási API-t, és
repülés-vezérlési capability nem regisztrálható.
